# NeoUniTextCollection
**A collection of texts about Neo Universe(Umamusume) in the game**  
  
I'm a Japanese. English is not my first language so there may be some mistakes. Suggestions or corrections are always welcome. Thank you for your understanding!  

## Project Structure
```
NeoUniverse					: text mainly from NeoUniverse context
├─StoryText
│  ├─CharacterStories		: text from character stories
│  ├─CommonTrainingEvents	: text from common training events
│  ├─CostumeTrainingEvents	: text from racewaer training events
│  └─RandomTrainingEvents	: text from random training events
└─VoiceText					: voice text of NeoUniverse
```
